# Terrain Island

Island with Unity 5.

![Island](doc/island.gif)


## Prerequires

1. Git 2.9+
3. Unity 5.4+


## How to Play

Clone

```
git clone https://github.com/humbertodias/unity-island.git
```

On Unity 5, open the project folder: **unity-island**


## Controls

Mouse or Keyboard Arrows
